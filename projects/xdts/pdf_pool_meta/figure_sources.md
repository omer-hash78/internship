# XDTS Figure Sources

This file records public reference material used to inform redrawn in-house
technical figures in the XDTS PDF pool. The resulting XDTS diagrams are not
direct copies; they are simplified internal redraws created to keep a
consistent document style.

## AV_FL_NAV_CL_1001_Preflight_Checklist_R000_2026_04_10.pdf

- Figure title: `Release Inspection Layout`
- XDTS figure intent: UCAV preflight inspection callout layout for the nose bay,
  SATCOM fairing, external stations, and nose gear service point.
- Reference sources:
  https://commons.wikimedia.org/wiki/File:Eagle_1_UAV_Drawing.png
  https://commons.wikimedia.org/wiki/File:Drohne_Silhouette.svg
- Source notes:
  both references were reviewed for general UAV outer-mold-line proportions and
  silhouette treatment; the XDTS figure is a fresh line drawing with XDTS
  labels and inspection callouts.
- License notes:
  the Wikimedia source pages list these files as public-domain material.
